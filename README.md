# Expense Tracker

This is a simple expense tracker website to help you manage your finances.

## Features

*   Add Expenses: Easily add your expenses with descriptions and amounts.
*   View Expenses: See a list of all your expenses.
*   Categorize Expenses: Categorize expenses for better tracking.
*   Total Expenses: Get a summary of your total expenses.

## Getting Started

### Prerequisites

*   Web browser

### Usage

1.  Open the `index.html` file in your web browser.
2.  Add your expenses using the form.
3.  View the list of expenses and the total amount.
